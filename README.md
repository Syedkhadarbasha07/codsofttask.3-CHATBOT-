# 🤖 Rule-Based Chatbot (Python)

## 📌 Project Description
This project implements a **simple rule-based chatbot** using Python.  
The chatbot responds to user inputs based on predefined rules using **if-else statements** and basic pattern matching techniques.

The project demonstrates fundamental concepts of:
- Natural Language Processing (NLP)
- Conversation flow design
- Rule-based Artificial Intelligence

---

## 🎯 Objective
To build a chatbot that can simulate basic human conversation by identifying user queries and providing predefined responses.

---

## 🚀 Features
- Rule-based responses using if-else conditions
- Interactive command-line chatbot
- Keyword and pattern recognition
- Date and time responses
- Exit command to end conversation

---

## 🛠️ Technologies Used
- Python 3

---

## ⚙️ How It Works
1. The user enters a message in the terminal.
2. The chatbot converts input into lowercase text.
3. The program checks predefined conditions.
4. A matching response is returned.
5. If no rule matches, a default reply is shown.
